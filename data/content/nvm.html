Install Nvm

Install Nvm Using Curl

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
Install Nvm using curl to bash shell
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | zsh
Install Nvm using curl to zsh shell

Install Nvm Using WGet

wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash
Install Nvm using wget to bash shell
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | zsh
Install Nvm using wget to zsh shell

Install Node Using Nvm

nvm install 14.16.1
Installs node version 14.16.1

Listing Versions

nvm ls
See what versions are installed
nvm ls-remote
See what versions are available to install
