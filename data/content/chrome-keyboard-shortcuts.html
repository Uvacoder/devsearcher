Address Bar

⌘ LGo to the address bar
⌃ F5Move your cursor to the address bar
Type a term ReturnSearch with your default search engine
Type a search engine TabSearch with a different search engine
Type a web address ReturnOpen a website in a background tab
⌃ ReturnComplete www and .com in the address bar for the current tab
⌃ ⇧ ReturnComplete www and .com in the address bar for a new tab

Window

⌘ NOpen a new window
⌘ ⇧ NOpen a new incognito window
⌘ ⌃ FToggle full screen mode
⌘ MMinimize the window
⌘ HHide the window
⌘ ⇧ WClose a window
⌘ QQuit Chrome

Bookmark, History and Tools

⌘ ⇧ BShow/Hide the Bookmarks toolbar
⌘ ⌥ BOpen the Bookmarks Manager
⌘ YOpen the History page
⌘ ,Open the Settings page
⌘ ⇧ JOpen the Download page
⌘ ⌥ JOpen the Developer Tools
⌘ ⇧ DeleteOpen Clear Data ⌥s

Tabs

⌘ TOpen a new tab and move to it
⌘ 1, ⌘ 2, ⌘ 3(etc up to 8)Select a tab 1 through 8
⌘ 9Select the last tab
⌘ ⇧ TReopen previously closed tabs in the order you closed them
⌘ [ or ←Open the previous page in your history in the current tab
⌘ ] or →Open the next page in your history in the current tab
⌘ ⌥ →Move to the next open tab
⌘ ⌥ ←Move to the previous open tab
⌘ WClose a tab

Page Navigation

⌘ ⇧ RReload and ignore cache
⌘ +Zoom in
⌘ -Zoom out
⌘ 0Reset zoom
SpaceMove down a screen
⇧ SpaceMove up a screen
⌘ SSave the page
⌘ DSave the page as a bookmark
⌘ ⇧ DSave open tabs in a new bookmarks folder
⌘ PPrint the page
